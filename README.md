🚗 Atividade: Sistema de Gestão de Frotas 🚗

👨‍💻 Descrição da Atividade:
Você deverá criar um Sistema de Gerenciamento de Frotas que irá apresentar o seguinte menu:

📋 Menu do Sistema:
1️⃣ – Cadastrar Novo Veículo
2️⃣ – Listar todos Veículos cadastrados
3️⃣ – Pesquisar um Veículo por placa
4️⃣ – Remover um Veículo
0️⃣ – Sair

🔧 Regras de Negócio:

🚫 Não permitir cadastrar veículos com os campos em branco.
🚫 Não permitir cadastrar veículos com ano "0".
🚫 Não permitir cadastrar veículos com o número de portas "0".
🚫 Não permitir o cadastro de dois veículos com a mesma placa.
🛠️ Estrutura do Sistema:
O sistema deverá ter no mínimo duas camadas:

Camada "View" (interface com o usuário)
Camada "Service" (lógica de negócios)
🔄 Dicas Importantes:

Limpe a tela a cada ação para uma melhor experiência do usuário.
Controle possíveis erros de digitação nas opções do menu.
💡 Objetivo: Desenvolva o sistema seguindo as boas práticas e garantindo que as regras de negócio sejam cumpridas.

🔍 Atenção: Certifique-se de que o sistema lida corretamente com entradas inválidas e possíveis erros! Boa sorte! 💪
